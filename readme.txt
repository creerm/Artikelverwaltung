Kompilierung
======================
Um das Programm zu starten kann das Shellskript "makefile.sh" ausgeführt werden.
Innerhalb des Shellskripts werden automatisch zwei befüllte csv-Dateien übergeben, aus denen gelesen wird und in welche geschrieben wird.

Bedienungshinweise
======================
In dem Fenster "Zugang/Abgang/Wareneingang" gibt es im unteren Teil eine Statusleiste, die über den Erfolg oder den Misserfolg  einer Aktion informiert.
Wird das Programm durch das 'x' in der oberen rechten Ecke geschlossen, werden die Veränderungen an den Datensätzen gespeichert.
